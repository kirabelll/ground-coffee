# Ground Coffee

A modern coffee shop web application built with Next.js, showcasing premium coffee experiences and connecting coffee enthusiasts.

## Features

- Modern, responsive design
- Coffee menu and product showcase
- Interactive user experience
- Optimized performance with Next.js App Router

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnmp dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a modern font family for optimal readability.

## Tech Stack

- **Framework:** Next.js with App Router
- **Styling:** CSS Modules / Tailwind CSS
- **Font:** Geist font family
- **Package Manager:** Bun (recommended)

## Development

The project structure follows Next.js App Router conventions:

- `app/` - Main application directory
- `app/page.tsx` - Home page component
- `app/layout.tsx` - Root layout component
- `app/globals.css` - Global styles
- `public/` - Static assets
